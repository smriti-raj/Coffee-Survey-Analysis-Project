# Coffee-Survey-Analysis-Project
📌 Overview

This project analyzes coffee consumption habits using survey data.
It includes data analysis, visualization, machine learning, and Power BI dashboards.

🎯 Objectives
Understand coffee consumption patterns
Analyze demographic behavior
Build a predictive model for coffee preferences                                   
Create interactive dashboards
🛠️ Tools & Technologies
Python (Pandas, NumPy, Seaborn, Matplotlib)
Jupyter Notebook
Power BI
Scikit-learn
📊 Key Insights
Most people drink 2–3 cups per day
Espresso-based drinks are most popular
Professionals consume more coffee
Spending correlates with brewing method
🤖 Machine Learning Model
Model: Random Forest Classifier
Target: Favorite Coffee Drink

Accuracy: ~70–80% (depending on cleaning)
📈 Dashboard Features
Coffee consumption trends
Demographic insights
Spending behavior
Interactive filters
